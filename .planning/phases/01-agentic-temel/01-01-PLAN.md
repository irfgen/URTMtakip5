# Phase 1 Plan: Agentic Temel Altyapı

## Overview

action-definitions.json dosyasını master agent'a entegre et, /api/master/consult endpointi oluştur, modül danışma modülü yaz.

## Tasks

### Task 1.1: action-definitions.json Entegrasyonu
**Verify:** action-definitions.json dosyası mevcut ve doğru yapıda

- [ ] `backend/multi-agent/action-definitions.json` dosyasını okuyan bir modül yaz
- [ ] JSON şemasını doğrula
- [ ] Modül agent'lara bu dosyayı yükle

**Files:**
- `backend/multi-agent/action-loader.js` — action definitions yükleyici

### Task 1.2: /api/master/consult Endpointi
**Verify:** curl ile consult endpointine ulaşılabiliyor

- [ ] `POST /api/master/consult` endpointi ekle
- [ ] Endpoint: module, action, context, alternatives_requested
- [ ] action-definitions.json'a göre onay gerektirip gerektirmediğini kontrol et
- [ ] Onay gerekliyse master'a yönlendir, gerekli değilse direkt işle

**Files:**
- `backend/multi-agent/api-server.js` — güncellenmiş API server

### Task 1.3: Modül Danışma Modülü (consult-master.js)
**Verify:** Modül ajanı master'a danışabiliyor

- [ ] `consult-master.js` modülü yaz
- [ ] REST POST /api/master/consult çağrısı
- [ ] WebSocket üzerinden yanıt dinleme
- [ ] Timeout yönetimi (30 sn)
- [ ] Bekleme modu implementasyonu

**Files:**
- `backend/multi-agent/consult-master.js` — modül danışma modülü

### Task 1.4: Master Agent Güncellemesi
**Verify:** Master agent action definitions kullanabiliyor

- [ ] `master-agent.js` güncelle
- [ ] action-definitions.json'u yükle
- [ ] Her aksiyon için onay mantığı ekle
- [ ] Alternatif aksiyon seçeneği sunma
- [ ] WebSocket yanıt gönderimi

**Files:**
- `backend/multi-agent/master-agent.js` — güncellenmiş master agent

## Verification

1. `node api-server.js` başlatılabiliyor
2. `curl -X POST http://localhost:3001/api/master/consult -H "Content-Type: application/json" -d '{"module":"stok_kartlari","action":"siparis_tetikle","context":{"stok_id":5}}'` çalışıyor
3. Onay bekleyen aksiyon master'dan yanıt bekliyor
4. Timeout sonrası modül bekleme moduna giriyor

## Success Criteria

- [ ] action-definitions.json doğru yükleniyor
- [ ] /api/master/consult endpointi çalışıyor
- [ ] Modül ajan master'a danışabiliyor
- [ ] Master onay verebiliyor veya alternatif öneriyor
